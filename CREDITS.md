# Credits

## Immersive Gambling

An Immersive Engineering addon for industrial coin minting and gambling machines.

---

## Original Work

### 777 Mod
**Author:** phaic
**License:** Apache 2.0
**Source:** https://modrinth.com/datapack/777

The original 777 mod by phaic provided the foundation for Immersive Gambling, including:

- Slot machine entity and animation system
- Blackjack table multiplayer game
- Gambling mechanics and reward distribution
- Click detection framework using armor stands and interactions
- Custom model data integration

We are grateful to phaic for creating and sharing this work under an open license.

---

## Derivative Work

### Immersive Gambling
**Authors:** zhintze, mooStack
**License:** Apache 2.0

This mod extends the original 777 concept with:

- Integration with Immersive Engineering machinery
- Industrial coin minting system using Metal Press and Alloy Kiln
- New coin types with multiplier mechanics
- Apotheosis and Silent Gear loot integration
- Realistic casino-style odds system
